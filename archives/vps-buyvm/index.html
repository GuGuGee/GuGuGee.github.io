1. 前言

BuyVM 成立于 2010 年，英国注册公司 FRAN­TECH SO­LU­TIONS LTD (07743806) 旗下的 VPS 品牌，属于知名老牌商家。

目前主营美国纽约 / 拉斯维加斯 / 迈阿密和卢森堡的 KVM Slice VPS，采用 KVM 虚拟，Stal­lion 控制面板 (自主研发)，1Gbps 带宽不限制流量，1 个 IPv4 和 /48 IPv6，免费快照。

BuyVM 在以上地区都提供了可供 VPS 挂载的 Block Stor­age Slabs (附加存储块)，每 256GB 仅需 $1.25 / 月，每月加 3 美元可以获得 DDoS 高防 IPv4 。

支持支付宝、Pay­Pal、信用卡等付款方式。

1G 内存及以上 VPS 支持 Win­dows 系统。

BuyVM 是自建网盘存储，BT/PT 离线下载，OneDrive、Google Drive 网盘中转离线下载的绝佳选择之一。

1.1 BuyVM官方网站

官网地址：https://buyvm.net (进入后会跳转到 fran­tech 域名的页面。因为是子品牌，所以采用了同一个销售和管理页面)

管理页面：https://my.frantech.ca

2. 特点

  • 无限流量！
  • 提供了可供 VPS 挂载的 Block Stor­age Slabs (附加存储块)
  • 卢森堡地区抗DCMA版权投诉，适合BT、PT下载
  • 每月加 3 美元可以获得 DDoS 高防 IPv4
  • 1G 内存及以上 VPS 支持 Win­dows 系统。
  • Stal­lion 控制面板 (自主研发)

3. 注意事项

  • 支付宝支付有优惠（看起来是官网是美元，实际上是加拿大元，会便宜一些）

  • 每个VPS支持挂载8个块存储，每个块存储最大支持10T硬盘

  • 商家开启了反欺诈验证，不要挂代理和魔法爱国科学上网，尽量使用真实信息下单(主要是注册邮箱和支付工具邮箱要对应，大概地址要和 IP 所在地相符)。

  • 推荐使用支付宝付款，加拿大元(CAD)结算便宜很多。比如原价是 5 USD ，PayPal 支付需要支付人民币 32 元左右，而支付宝结算是 5 CAD 结汇成人民币，就只需要 25 元左右。

  • BuyVM 的价格是按自然月结算的，比如当月1日购买就是一整个月的价格，2-15日为当月剩余天数计算的价格，15日后购买就是按当月剩余天数计算的价格加下个月整月的价格。

  • PayPal 付款需注意注册邮箱一致，否则可能风控进入人工审核。

  • 附加存储块购买后要进行挂载操作才能附加到 VPS 上，教程随便咕鸽搜索都能找到。

  • 由于非常抢手，可能存在 VPS 有货而存储块没货，或者 VPS 没货而存储块有货的情况。总之有货的先买，等几天补货时再来买就凑齐了。

  • 补货时间一般在工作日的凌晨1-2点(北京时间)左右，提前注册账号、登录账号再抢购。

  • 对于下载用途，建议不要长时间大量占用网络，适当进行限速，以免被限速或停机。

  • 卢森堡(LU)机房抗 DCMA 版权投诉，适合需要使用 BT 下载大量生肉资源的小伙伴。

  • 拉斯维加斯(LV)机房可同时兼顾魔法科学爱国上网与下载需求。

  • 美国对版权要求严格，拉斯维加斯(LV)、纽约(NY)、迈阿密(MIA)机房的 VPS 不要用于下载国外知名 BT 资源网站(比如rarbg)提供的热门影视资源，以防被钓鱼执法导致停机。

  • 512M 内存配置套餐已于2021年11月永久下架，表格中的购买链接仅供纪念。

4. 适合人群

  • BT、PT用户
  • 不适合建站（个人感觉）
