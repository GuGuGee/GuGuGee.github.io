1. 前言

熟悉咕咕的小伙伴应该都比较清楚，咕咕每次搭建项目的服务器都没有用国内的机器，原因有三点：

1、国内的机器都必须备案才能使用web服务。

咕咕懒，所以就用的国外的VPS了。

2、国内的机器默认和你家的网络一样，都无法直接访问GitHub。

这会导致咕咕介绍的很多脚本都无法下载（因为代码98%都在GitHub上），同样，国内的机器也无法直接获取到最新的docker镜像，因为也被墙了。如果想要正常下载GitHub或者docker hub上的内容，你需要换源（而很多时候，国内源并不能及时更新，常常会导致一些问题，e.g :aliyun mirror 更新不及时。比如 Spring Boot 3.0 发布了，aliyun mirror 得好几天才能同步好，这段时间内我们都无法正常下载依赖。）。

3、国内机器的带宽普遍比较小，通常只有几兆。

所以咕咕一开始首推的是香港的腾讯云轻量应用服务器，当时一个月只要24块钱，而且还是30M的带宽，电信CN2 GIA回程，而且因为是在香港（非大陆），访问GitHub和Docker Hub丝毫没有问题，属于是性价比非常高的机器了。

然而，随着腾讯产品线的升级，24元/月的套餐已经一去不复返，现在官网在售的最便宜的套餐是：

32元/月的，其实也还可以，但是好像已经没货很久很久了。
现在能买的直接到了100元/月，这别说对于刚玩VPS的小伙伴，就算是咕咕也不舍得买 = =

当然我们可以选择买别的区域的机器，但是在可选的非大陆服务器里面，香港毕竟是里大陆物理距离最近的（虽然不一定延时最低）。

有很多小伙伴反馈就是像要建个1 IP的小博客，不想备案又想速度快，还不想花费太多金钱在上面，问咕咕有没有其他的香港的机器推荐。

今天我们就来介绍一个新的商家——莱卡云。

2. 总体印象

商家给咕咕开了两台机器用于测试（分别是香港BGP和三网香港CN2 GIA）。

近一个月时间使用下来，咕咕感觉还行，这边做一个简单的Review。

2.1 优点

  • 弹性，扩容方便
  • 后台管理界面清晰易上手
  • 延时优秀，尤其是CN2 GIA与国内基本无差别
  • 香港不需要备案
  • 客服反应迅速
  • 支持月付、季付、半年付、年付
  • BGP套餐送免费快照一个
  • BGP解锁美区奈飞

2.2 缺点

  • 带宽太低
  • 需要实名
  • 磁盘IO比较低（应该是商家锁了IO）
  • gia 不解锁奈飞
