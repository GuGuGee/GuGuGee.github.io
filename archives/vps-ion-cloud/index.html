1. 前言

今天来介绍一下咕咕的这台香喷喷的年付20.19美元的服务器——iON

监控搭建：教程

2. 简介

iON属于是Krypt旗下，Krypt又是属于VPLS Inc公司旗下，提供CN2 GIA、CN2 GT、PCCW、中国移动直连等多种线路套餐，同时提供Linux和Windows系统可选。建站非常稳定。

总体来说，是一家大公司，大家不用担心跑路风险。

而且iON专门做了中文的网站，方便国人购买VPS。

2.1 优点

咕咕实际使用了这么久，总结了iON的几个优点，供大家参考：

  • 云服务器流量仅计算出站方向（即单向流量而不是一般服务器商家普遍的双向流量，这个是VPS商家中极少见的，意味着1T就是真正的1T）
  • 大厂（稳定，不用担心跑路）
  • 支持支付宝、PayPal付款购买
  • 客服 7×24小时 反馈快（可以直接中文交流）
  • 洛杉矶、圣何塞三网回程AS4837（咕咕实测晚高峰依旧丝滑，南方电信甚至有时候跑的比搬瓦工DC06还高）

2.2 缺点

  • 贵
  • iON无自带快照和备份功能（需要自己备份，备份教程推荐）
  • 每台机器只能换一次IP，需要花费2美元

4. 注意事项

4.1 不要轻易DD系统！！！可能会删鸡！

4.2 关于IP被墙

新买的机器，如果发现IP被墙了，可以在24小时之内联系客服免费更换。

超过24小时之后，每台机器只有一次换IP的机会，而且需要付2美元的费用。

4.3 流量超了怎么办？

如果你使用的流量超出限额，你的VPS将被暂停（会有邮件通知），然后你可以加购流量包，每250GB流量5美元（别问我怎么知道的，1T流量一般人是用不完的，我是被打了 = =）
